# keenotes-releases

The public repository for releasing keenotes to public users.

If you want to know more about keenotes, check out <https://keenotes.afoo.me>

It's a fleeting-idea-capture app, available for all types of devices(almost), from win/linux/macos to ios/android.

No Web, since we have not found a proper and safe way to do E2EE with browsers yet.
